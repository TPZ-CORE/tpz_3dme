
# TPZ-CORE 3DME

## Requirements

1. TPZ-Core: https://github.com/TPZ-CORE/tpz_core
2. TPZ-Characters: https://github.com/TPZ-CORE/tpz_characters

# Installation

1. When opening the zip file, open `tpz_3dme-main` directory folder and inside there will be another directory folder which is called as `tpz_3dme`, this directory folder is the one that should be exported to your resources (The folder which contains `fxmanifest.lua`).

2. Add `ensure tpz_3dme` after the **REQUIREMENTS** in the resources.cfg or server.cfg, depends where your scripts are located.
